RTB5联盟返利报表API接口说明及调用规范

接口调用规范

规范约定了调用接口的常规总则，在具体的接口中，如无特别的说明，请按规范调用。

接口协议：HTTP GET

编码格式：UTF-8，对于输入和输出同是utf-8;

API参数说明
token 字符串（必填），如“NDI5NDkwMTc3MQ==” 网站密匙，向管理员获取
begin_time/end_time 查询的时间段（选填） 开始时间，结束时间
格式：unix时间
status 单独查询的订单类型（选填） 0未处理，1待生效，2无效，3已删除，4已生效,5商家修正
tags 自定义标签,链接中自带的标签，方便网站主统计分析（选填）

接口地址：

http://121.40.129.177:8090/order.php

示例链接：

http://121.40.129.177:8090/order.php?token=NDI5NDkwMTc3MQ==&tags=321
